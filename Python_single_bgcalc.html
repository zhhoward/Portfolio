Using Jupyter Notebooks through Anaconda Navigator & Python

GitHub files.

With yfinance in Jupyter Notebooks I created a simple Ben Graham Calculator using current Book Value and Earnings per Share. All information is provided for informational and research purposes only.

It takes a case insensitive input and, if it is a valid ticker on the Nasdaq, produces an Intrinsic Value for that stock.

In this example we analyse Kodak stock, ticker symbol KODK. As you can see Ben Graham would consider this stock undervalued.

Analysing Facebook stock, ticker symbol META. With a price double of its intrinsic value, Ben Graham would consider this stock overvalued.

Analysing Walmart stock, ticker symbol WMT. With a Price to Earnings ratio over 30, Ben Graham would consider this stock overvalued.
